# ADR-0002: Voyage AI for Embeddings

## Status
Accepted

## Context
Need high-quality embeddings for semantic search. Options: OpenAI, Cohere, Voyage AI.

## Decision
Use Voyage AI `voyage-3-large` (1024 dimensions) for embeddings.

## Consequences
**Positive**: State-of-art retrieval quality, good pricing
**Negative**: Vendor dependency

2026-01-20
